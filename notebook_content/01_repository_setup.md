# Part 1: Repository Setup (5 min)

## Step 1: Fork the Repository

Before we begin in Colab, you need your own copy of the conference-rag repository:

1. Go to: **https://github.com/byu-cs-452/conference-rag**
2. Click **"Fork"** in the top right corner
3. Under "Repository name", keep it as `conference-rag` (or rename if you prefer)
4. **Uncheck** "Copy the `main` branch only" if prompted (we only have `main`, but just in case)
5. Click **"Create fork"**

You now have your own copy at: `https://github.com/YOUR-USERNAME/conference-rag`
